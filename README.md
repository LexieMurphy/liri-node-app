# liri-node-app

In this CLI LIRI app the user can access Spotify song data via the command line node. It takes in certain parameters and returns some selected results based on the api criteria for up to 10 results.

Simply use the command line and type the following:
    _node .\liri.js spotify-this-song **"Let It Be"**_

![Command Line Screen Shot](.\images\CommandLineSS1.png)

The end of the command line within the quotes will be the selected song to return results on.
